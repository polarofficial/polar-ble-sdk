  • requestRecordingStatus(_:)

    Request current recording status. Supported only by Polar H10. Requires polarFileTransfer feature.

  • fetchStoredExerciseList(_:)

    Api for fetching stored exercises list from Polar H10 device. Requires polarFileTransfer feature. This API is working for Polar OH1 and Polar Verity Sense devices too, however in those devices recording of exercise requires that sensor is registered to Polar Flow account.

  • fetchExercise(_:entry:)

    Api for fetching a single exercise from Polar H10 device. Requires polarFileTransfer feature. This API is working for Polar OH1 and Polar Verity Sense devices too, however in those devices recording of exercise requires that sensor is registered to Polar Flow account.

  • removeExercise(_:entry:)

    Api for removing single exercise from Polar H10 device. Requires polarFileTransfer feature. This API is working for Polar OH1 and Polar Verity Sense devices too, however in those devices recording of exercise requires that sensor is registered to Polar Flow account.

  • startListenForPolarHrBroadcasts(_:)

    Start listening to heart rate broadcasts from one or more Polar devices

  • requestStreamSettings(_:feature:)

    Request the stream settings available in current operation mode. This request shall be used before the stream is started to decide currently available settings. The available settings depend on the state of the device. For example, if any stream(s) or optical heart rate measurement is already enabled, then the device may limit the offer of possible settings for other stream feature. Requires polarSensorStreaming feature.

  • requestFullStreamSettings(_:feature:)

    Request full steam settings capabilities. The request returns the all capabilities of the requested streaming feature not limited by the current operation mode. Requires polarSensorStreaming feature. This request is supported only by Polar Verity Sense firmware 1.1.5

  • startEcgStreaming(_:settings:)

    Start the ECG (Electrocardiography) stream. ECG stream is stopped if the connection is closed, error occurs or stream is disposed. Requires polarSensorStreaming feature. Before starting the stream it is recommended to query the available settings using requestStreamSettings

  • startAccStreaming(_:settings:)

    Start ACC (Accelerometer) stream. ACC stream is stopped if the connection is closed, error occurs or stream is disposed. Requires polarSensorStreaming feature. Before starting the stream it is recommended to query the available settings using requestStreamSettings

  • startGyroStreaming(_:settings:)

    Start Gyro stream. Gyro stream is stopped if the connection is closed, error occurs during start or stream is disposed. Requires polarSensorStreaming feature. Before starting the stream it is recommended to query the available settings using requestStreamSettings

  • startMagnetometerStreaming(_:settings:)

    Start magnetometer stream. Magnetometer stream is stopped if the connection is closed, error occurs or stream is disposed. Requires polarSensorStreaming feature. Before starting the stream it is recommended to query the available settings using requestStreamSettings

  • startOhrStreaming(_:settings:)

    Start OHR (Optical heart rate) PPG (Photoplethysmography) stream. PPG stream is stopped if the connection is closed, error occurs or stream is disposed. Requires polarSensorStreaming feature. Before starting the stream it is recommended to query the available settings using requestStreamSettings

  • startOhrPPIStreaming(_:)

    Start OHR (Optical heart rate) PPI (Pulse to Pulse interval) stream. PPI stream is stopped if the connection is closed, error occurs or stream is disposed. Notice that there is a delay before PPI data stream starts. Requires polarSensorStreaming feature.

  • enableSDKMode(_:)

    Enables SDK mode. In SDK mode the wider range of capabilities is available for the stream than in normal operation mode. SDK mode is only supported by Polar Verity Sense (starting from firmware 1.1.5). Requires polarSensorStreaming feature.

  • disableSDKMode(_:)

    Disables SDK mode. SDK mode is only supported by Polar Verity Sense (starting from firmware 1.1.5). Requires polarSensorStreaming feature.
